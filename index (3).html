Angular

um Framework. Um frame é uma estrutura de trabalho. Basicamente, é uma ferramenta que permite aos desenvolvedores trabalhar com mais eficiência e de forma mais organizada.

O que ela faz por você?

1. vantagem de ser baseado no TypeScript da Microsoft, que é um sistema de transpilação JavaScript

2. O TypeScript permite ao programador explorar funções anônimas, genéricas, todas em uma tipagem estática.

3. escolha do mecanismo de atualização de dados vinculados, entre o modelo e a visualização MVC, do tipo “vinculação unilateral” e não “vinculação bidirecional”

Oque o angular ajuda?

O Angular pode ser utilizado para criar aplicações web SPA, aplicações mobile (com o suporte do Ionic, por exemplo) ou até mesmo aplicações desktop (com o suporte do Electron).
